基本的python接口
================

在易度中，出于安全的考虑，对Python的一些常用包做了控制：

Python的一些常用包
--------------------

- 已import的包	
    datetime, calendar, xlwt, StringIO, 

- 可import的包	

    - 'time', 时间
    - 'random', 随机
    - 'hashlib', hach函数
    - 'urllib2', http通信
    - 'Crypto.Cipher', 加密
    - 'array', 阵列
    - 'binascii', binascii转换
    - 'xmlrpclib'：xmlrpc通信

基本函数
------------

- dict, set, list,buffer, sum, type, base64,

-----------

- str2int
- str2float,

- objectProvides,
- getUtility

----------------

- json , json格式的数据解析
